Borrowing a Book

  1. Go to the library and ask the librarian for the book you want to borrow.
  2. Receive the book and scan it using the self-issuing kiosk.
  3. Enter your student credentials to complete the borrowing process.
  4. The book will be registered under your name in the library database.
  5. You will receive an email confirmation specifying the return date.

Returning a Book

  1. Drop the book in the return book slot.
  2. The system will scan the book to verify its return status.
  3. The system will check if the book was returned on time.
  4. Return confirmation will be updated in the database.
